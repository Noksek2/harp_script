# Harp Script

- 개발 기간: ~2018년 12월

- 상태: 실험작 / 개발 중단

## 소개

Harp Script는 2018년경 개발된 간단한 실험용 스크립트 언어입니다.

## 특징

- 미약한 기능

- 간단한 수학 연산

- 콘솔 입출력

- dxlib 관련 일부 내장 함수

## 구현 특징

- AST 없는 바이트코드 생성: AST(추상 구문 트리)를 명시적으로 생성하지 않습니다. 재귀 하강 파서가 연산자 우선순위 스택을 이용해 구문 분석과 동시에 바이트코드를 직접 생성합니다. (`express.cpp` 참고)
- 차량기지 알고리즘 (shunting yard algorithm) 사용 - 굳이 재귀하향 + 차량기지 둘 다 쓰는 바람에 괜히 늦어지는데 이때는 잘 몰라서 걍 넣음.
- 간단한 FFI (Foreign Function Interface):** WinAPI(`LoadLibrary`/`GetProcAddress`)를 사용해 `.dll` 파일(예: `DxLib.dll`)을 동적으로 로드합니다. 이 함수들을 내장 함수(infunc)로 래핑하여 바이트코드 VM에서 직접 호출 (`memory.h`, `infunc.cpp` 참고)
- 스택 기반 VM : C++로 구현된 간단한 스택 기반 가상 머신이 바이트코드를 해석하고 실행합니다. (`execute.cpp` 참고)
- 부실해도 기본에 충실함
  
## 사용법

1. a.harp 파일에 코드를 작성합니다.

2. harp.exe를 실행합니다.

3. 실행 결과로 바이트코드 파일이 생성됩니다.

4. 콘솔, 프로그램 실행을 통해 결과를 확인 가능합니다.
