# Harp Script

- 개발 기간: ~2018년 12월

- 상태: 실험작 / 개발 중단

## 소개

Harp Script는 2018년경 개발된 간단한 실험용 스크립트 언어입니다.

## 특징

- 미약한 기능

- 간단한 수학 연산

- 콘솔 입출력

- dxlib 관련 일부 내장 함수

## 사용법

1. a.harp 파일에 코드를 작성합니다.

2. harp.exe를 실행합니다.

3. 실행 결과로 바이트코드가 생성됩니다.
